An HTTP Webserver. Created as part of the 42 Curriculum.

* Handles multiple clients HTTP request.

* Allows for downloading files.

* Can set up a server with multiple listening sockets.

* Handles the GET, POST, DELETE request.

* Provides simple CGI which creates a file on the server and deletes a file from the server.

* Is routing in accordance to an URL.

* Has a configure file.

* Created HTML pages for an Internet in 2003 expierience.

* Allows file upload, both txt files and jpg files. (in progress).
