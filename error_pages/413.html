Error 413 - Request Entity Too Large

413

Yikes! The request entity is too large for the server to process.

Possible reasons for this error include:

  • The client attempted to upload a file or data that exceeds the server's specified limit.
  • The server's configuration does not allow requests of this size.

Please try the following:

  • Reduce the size of the request entity or the file you are trying to upload.
  • Contact the website administrator to inquire about the maximum allowed size for requests.
